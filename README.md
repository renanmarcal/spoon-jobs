# job pipeline

Dashboard pessoal para acompanhamento de processos seletivos.

## o que faz
- Kanban de vagas por etapa (aplicado, triagem, entrevista, proposta, etc.)
- Registro de comentários e histórico por vaga
- Favoritos e filtros
- Export/import dos dados em JSON

## acesso
Protegido por senha. Dados salvos no localStorage do navegador.
Para migrar entre dispositivos: exportar JSON → importar no novo navegador.

## stack
HTML/CSS/JS — arquivo único, sem dependências de build.
Hospedado via GitHub Pages.

## atualizar
Editar `index.html` direto pelo GitHub e commitar. O Pages atualiza em ~1 min.
